Collection of methods to help manipulate maps.

The general flow is to input a map, do a bunch of transformations, and output the transformed map at the end. Note that the input map should never be used again. This class does not make any guarantee that it will / will not modify the input map.

Example:

let map = new Map<number, number>([[1, 2], [3, 4]]);
Maps
    .of(map)
    .forEach((value: number, key: number) => {
      console.log(value + key);
    });

Note that every key and value in the map must be of the same type.
