Collection of methods to help manipulate sets.

The general flow is to input a set, do a bunch of transformations, and output the transformed set at the end. Note that the input set should never be used again. This class does not make any guarantee that it will / will not modify the input set.

Example:

Set
    .fromArray([1, 2, 3])
    .data;  // Set([1, 2, 3])

Note that every value in the set must be of the same type.
