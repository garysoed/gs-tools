Utility to make HTTP requests.

Base use case is to start with the Http object, set all the parameters, and call send. The send will send the request and return a Promise. For example:

import Http from './net/http';

Http.post('http://path')
    .setFormData({
      'cost': '$2',
      'food': 'ice cream'
    })
    .send()
    .then((response: string) => {
      // Do something with the response
    });
