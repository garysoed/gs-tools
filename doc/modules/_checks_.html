Checks:object checks.ts:18 code

Utility to do various dynamic time type checks.

In addition to supporting dynamic type checks, this uses Typescript's type guards to cast the values. For example:

if (Checks.isArrayOf<string>(input, String)) {
  // It's safe to treat input as string[] in this block.
}

Use this library whenever you want to narrow down a type, either due to union types or due to type inheritance.

Checks if the given value is an instance of the given type.

This handles primitives like string, number, boolean by treating them as their non primitive versions. I.e.: String, Number, and Boolean.
