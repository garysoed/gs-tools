Attribute directive to apply BEM to CSS classes.

Using this consists of two parts:

  1. Use gs-bem-class instead of ng-class or class on the elements you want to apply this to.
  2. At the root of your directive, add a gs-bem-root attribute.

This directive will replace the CSS classes specified in gs-bem-class by prepending the value in gs-bem-root.

<div gs-bem-root="root-elem">
  <ul>
    <li gs-bem-class="list-item">
      This tag will have CSS class: root-elem__list-item
    </li>
  </ul>
</div>

The value of the attribute can take the following possible forms:

  1. Simple string: This string will simple be prepended with the gs-bem-root value.
  2. Array of strings: All of the string elements in the array will be prepended with the gs-bem-root value.
  3. Record of booleans: If the value of the entry is true, the key of that entry will be applied, with the gs-bem-root value prepended to the key.
